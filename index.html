pins.js: A cross-language data sharing library#

The pins.js library is implemented in JavaScript making it ideal to read and write datasets from plain HTML or Node.js applications. It provides support for Python as well by embedding pins.js into its Python package. Since pins.js was ported from the pins R package, it can also interoperate with R, making it an ideal library for sharing datasets across R, Python and JavaScript.

Pin remote resources locally with pin(), work offline and cache results.
